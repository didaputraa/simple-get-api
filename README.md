# Cara operasi

1. buka cmd/terminal dan arahkan root ke folder aplikasi
2. ketik `composer install` dan tunggu hingga proses installasi selesai
3. kemudian edit file `.env` dan sesuaikan spesifikasi server anda
contoh
    `DB_DATABASE=testing_guestbook`
    `DB_USERNAME=root`
    `DB_PASSWORD=`

4. buat database mysql sesuai nama yg ada di file `.env` 
5. import database data pada file `testing_guestbook.sql`
6. selesai
